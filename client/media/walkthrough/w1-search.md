### Search for ABAP Objects

**Ctrl/Cmd+Shift+P** → **ABAP FS: ABAP Search for object** (or **Ctrl+Alt+;**)

**First time:** Multi-select type picker (Classes, Programs, Tables, etc.). Save preference to skip next time.

**Search:** Type 2+ characters with wildcards:
- `Z*ORDER*` — custom objects with "ORDER"
- `CL_BCS*` — classes starting with CL_BCS
- `BAPI_USER*` — user-related BAPIs

**Tips:**
- Filter button (top-right of search box) to change types anytime
- Right-click objects in Explorer → **Add to Favorites**
