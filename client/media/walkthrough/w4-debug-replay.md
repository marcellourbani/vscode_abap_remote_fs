### Record & Replay Debug Sessions

**Record:**
1. Start ABAP debug session (see Debugging in Views & Tools)
2. **Ctrl/Cmd+Shift+P** → **ABAP Replay: Start Debug Recording**
3. Debug — every step captured
4. **Ctrl/Cmd+Shift+P** → **ABAP Replay: Stop Debug Recording**
5. Save `.abaprecord` file

**Replay:** **Ctrl/Cmd+Shift+P** → **ABAP Replay: Replay Debug Recording**
- Step forward AND backward
- No SAP connection needed
- Share with teammates
