# Instructions for AI Agents

When contributing to this project, you MUST follow the guidelines in [CONTRIBUTING.md](CONTRIBUTING.md).

Key requirements:
- **Run the format script** (`npm run format`) after editing files.
- **Do not commit scratch files** or session notes.
- **Check your diffs** carefully for sensitive information.
- **Follow the code style** (no semicolons, etc.) as defined in [.prettierrc.json](.prettierrc.json).
